# Skills 目錄結構範例
``` t
blog-post-master/
├── SKILL.md             # 核心定義（必備）
├── scripts/             # 自動化腳本
│   └── seo_check.py     # 檢查關鍵字密度的 Python 腳本
├── templates/           # 輸出格式
│   └── post_layout.md   # 標準文章模板
└── references/          # 參考資源
    └── style_guide.md   # 品牌語氣規範
```

# SKILL.md 範例
``` t
---
name: blog-post-master
description: 當使用者需要撰寫技術文章、優化 SEO 或產生部落格草稿時使用此技能。
metadata:
  version: "1.0.0"
  author: "AI Expert"
---

# 技術部落格大師技能

## 📋 使用時機
- 使用者要求「幫我寫一篇關於...的文章」。
- 使用者需要針對現有文案進行 SEO 優化。
- 需要根據特定框架（如 AIDA）構思內容大綱。

```
