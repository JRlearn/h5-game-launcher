# 遊戲規則

## 遊戲簡介
1A2B（又稱 Bulls and Cows）是一款經典的數字猜測遊戲，玩家需要通過不斷猜測來找出隱藏的數字組合。

## 遊戲規則
1. 遊戲開始時，系統會隨機產生一組 **4 位不重複的數字**。
2. 玩家需要輸入 4 位數字進行猜測。
3. 系統會根據玩家的輸入給出回饋：
   - **A（Bulls）**：表示數字 **正確且位置正確**。
   - **B（Cows）**：表示數字 **正確但位置錯誤**。
4. 玩家根據回饋繼續猜測，直到完全猜中（4A0B）。
5. 記錄玩家的猜測次數，並計算完成遊戲所需的步數。

## 例子
### 隱藏數字： **5271**
| 玩家輸入 | 結果 |
|----------|------|
| 1234     | 1A2B |
| 5678     | 2A1B |
| 5271     | 4A0B （正確答案！）|

## 遊戲模式
1. **單人模式**：挑戰自己，用最少的步數猜中數字。
2. **雙人對戰模式**：兩名玩家輪流猜測，最先猜中的玩家獲勝。
3. **計時模式**：在限定時間內猜中數字，測試反應速度。

## 策略與技巧
- **從 0-9 中選擇較平均分佈的數字作為初始猜測**。
- **根據 A 和 B 的數量來推理數字的可能位置**。
- **避免重複使用已經被排除的數字**。
